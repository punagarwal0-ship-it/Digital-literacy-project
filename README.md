## Project Overview
As a Student Digital Ambassador, I have developed this portfolio to help peers navigate the digital world. This repository contains resources for professional profile building, safe internet practices, and digital collaboration.

## Module Summaries
Based on the CSE0001 curriculum, this portfolio covers:

* **Module 1: Presentation & Spreadsheets:** Focuses on creating visual resources using Canva and Prezi, and managing data with Excel/Spreadsheets.
* **Module 2: Collaborative Platforms:** Utilization of Google Workspace (Docs, Sheets, Forms) and coding platforms like HackerRank and CodeChef for peer collaboration.
* **Module 3: Digital Portfolios:** Building a professional presence on LinkedIn, GitHub, Kaggle, and StackOverflow to connect with the academic and professional community.
* **Module 4: Digital Etiquette:** Mastering professional email communication, signature creation, and responsible social media behavior.
* **Module 5: Cyber Security:** Awareness of financial fraud, UPI scams, personal digital hygiene, and prevention measures.
